# BPHS Golang Backend

## Getting Started

### 1. Install Go (Golang)

- Download Go from [https://go.dev/dl/](https://go.dev/dl/)
- Verify installation:
  ```bash
  go version
  ```

---

### 2. Running the Application

- Install dependencies:
  ```bash
  go mod tidy
  ```
- Run the application:
  ```bash
  go run main.go
  ```
- Run using CompileDaemon:
  ```
  CompileDaemon -command="./bphs"
  ```

---

✅ This way, you’ll have a running Go backend.
